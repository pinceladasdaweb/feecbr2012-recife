A Regra viewport

Criada originalmente pela Apple para o Safari do iPhone

Adotada posteriormente por outros browsers

<meta name="viewport" content="width=device-width, initial-scale=1">

Não desabilite o zoom

<meta name="viewport" content="width=device-width, user-scalable=no">

<meta name="viewport" content="width=device-width, minimum-scale=1, maximum-scale=1">

O novo @viewport

Regra oficial criada pelo W3C

@viewport {
	width: device-width;
}

/* Prefixos */
@-ms-viewport { width: device-width; }
@-o-viewport  { width: device-width; }

Flexibilidade com Media Queries

As antigas Media Types

<link rel="stylesheet" href="styles.css" media="screen">
<link rel="stylesheet" href="mobile.css" media="handheld">

CSS3 media queries

<link rel="stylesheet" href="iphone.css" media="screen and (max-width: 320px)">

Pdemos também utilizar min-width, width e device-width e as mesmas variações pra height.

Media queries com base na orientação do device

//Quando o device estiver na vertical
<link rel="stylesheet" href="portrait.css" media="screen and (max-width: 320px) 
		and (orientation: portrait)">
        
//Quando o device estiver na horizontal
<link rel="stylesheet" href="landscape.css" media="screen and (max-width: 480px) 
		and (orientation: landscape)">

Media queries dentro de arquivos CSS

body { background: white; }
 
/* Aplicada quando a largura mínima for 320px */
@media screen and (min-width: 320px) {
   body { background: red; }
}
 
/* Aplicada quando a largura mínima for 480px */
@media screen and (min-width: 480px) {
   body { background: green; }
}

Responsive Images

Imagens que se adaptam automaticamente ao dispositivo

img {
     max-width: 100%;
     height: auto;
}

No Internet Explorer

img {
     width: 100%;
